# ⚡ Lenny Dardet - Portfolio Moderne

## 🚀 Fonctionnalités

- **Design Premium** : Effets de glassmorphism, gradients modernes et typographie soignée.
- **Animations Fluides** : Transitions douces et micro-interactions avec Framer Motion.
- **Single Page Application** : Navigation fluide par défilement (scrolling).
- **Responsive Design** : Entièrement adapté aux mobiles, tablettes et ordinateurs.
- **Dark Mode** : Palette de couleurs sombre élégante par défaut.
- **Data Driven** : Facilement modifiable via un fichier de configuration centralisé (`portfolioData.js`).

## 🛠️ Stack Technique

- **Framework** : [React 19](https://react.dev/)
- **Bundler** : [Vite 6](https://vitejs.dev/)
- **Styles** : [Vanilla CSS](https://developer.mozilla.org/fr/docs/Web/CSS) (Design System personnalisé)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Icônes** : [Lucide React](https://lucide.dev/) + SVG personnalisés

## 📦 Installation

Pour lancer le projet localement :

1. Cloner le dépôt :
```bash
git clone <url-du-depot>
```

2. Installer les dépendances :
```bash
npm install
```

3. Lancer le serveur de développement :
```bash
npm run dev
```

4. Ouvrir `http://localhost:5173` dans votre navigateur.

## 📁 Structure du Projet

```text
src/
├── assets/         # Images, CV et ressources statiques
├── components/     # Composants React (Hero, Skills, Projects, etc.)
├── data/           # Fichier centralisé des données (portfolioData.js)
├── App.jsx         # Composant racine et layout principal
├── index.css       # Design System et styles globaux
└── main.jsx        # Point d'entrée de l'application
```

## 🌐 Déploiement

Le site est optimisé pour un déploiement rapide sur **Vercel** ou **Netlify**.
Pour générer la version de production :

```bash
npm run build
```
